## ‚ú® Features

- **Real-time Analytics:** High-level metrics for Total Products, Inventory Value, and Low Stock alerts.
- **Inventory Health Donut:** Dynamic visual representation of stock distribution (Healthy vs. Low vs. Out).
- **Acquisition Trends:** Automated weekly bar charts tracking inventory growth using MongoDB timestamp logic.
- **Smart SKU Tracking:** Robust CRUD operations with duplicate key prevention and automated stock level coloring.
- **Modern UI/UX:** Responsive design featuring Flickering Grids, Glassmorphism, and Lucide icons.
- **Secure Access:** Authentication powered by Clerk to protect sensitive warehouse data.
